The Picdump page

From this page you can download a regression program and a statistical program written for Windows 3.1 and higher.
The program PICdump is especially designed to print Lotus PIC files on a printer. It can print 1 to 12 PIC files on a page.
The Lotus PIC files format is extremely simple and vector orientated and it is a pity that the Lotus cooperation dropped this file format in its newest releases.
PIC files can be viewed and edited, fonts and line thickness can be changed.

Download the PICdump program for Windows. Version 130895
